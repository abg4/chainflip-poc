## Chainflip <> Across PoC

## Usage

### Build

```shell
$ forge build
```

### Test

```shell
$ forge test --rpc-url {MAINNET_RPC_URL}
```

### Format

```shell
$ forge fmt
```

### Gas Snapshots

```shell
$ forge snapshot
```

### Anvil

```shell
$ anvil
```

### Cast

```shell
$ cast <subcommand>
```

### Help

```shell
$ forge --help
$ anvil --help
$ cast --help
```
